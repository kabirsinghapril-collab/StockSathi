# 📦 StockSathi

StockSathi is a full-stack Inventory & Sales Management SaaS built using:

- Next.js
- FastAPI
- Supabase
- Recharts

## Features

- Product Management
- Inventory Tracking
- Sales Recording
- Revenue Analytics
- Low Stock Alerts
- AI Inventory Assistant
- User Authentication
- Category Filtering
- Interactive Charts

## Tech Stack

Frontend: Next.js + TypeScript  
Backend: FastAPI  
Database: Supabase  
Authentication: Supabase Auth
